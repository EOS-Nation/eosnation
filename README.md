# EOS Nation

#### You can see a demo in here:

http://eosnation.carmel.io/

## Development setup

### Prerequisites:

You need to install chunky-cli

### Setup

`npm i`
Create web folder:
`mkdir .chunky; cd .chunky; touch web`

## Run chunky website

`chunky start web`
Open browser tab an type: `http://localhost:8082`
